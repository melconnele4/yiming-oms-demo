# 一鸣供应链订单中台 OMS Demo

> 温州一鸣食品供应链订单中台管理系统 — 多渠道订单统一管理平台功能演示

## 🌐 在线演示

**🔗 公网访问地址：[https://melconnele4.github.io/yiming-oms-demo/](https://melconnele4.github.io/yiming-oms-demo/)**

- 默认账号：`admin` / 密码：`123456`（Demo 模式，直接登录即可）

## ✨ 功能模块

### 📊 工作台
- 订单统计概览（今日订单数、已审核、待审核、订单金额）
- 订单趋势图表（近7日柱状图）
- 最新业务动态实时展示

### 📋 业务管理
- **订单审核** — 多维度筛选、批量审核、订单详情查看
- **订单新建** — 手工录入订单、物料添加、Excel 导入
- **订单查询** — 历史订单检索与履约进度跟踪

### ⚙️ 后台管理
- **截止时间规则** — 区域/客户/物料四维组合的要货提前天数与截止时间维护
- **附赠品规则** — 主物料与附赠品的绑定关系及数量配比维护
- **组合商品 BOM** — 套餐商品物料绑定关系，支持自动拆单

### 📈 数据中心
- **订单监控** — 全链路处理状态实时监控（要货端→OMS→WMS→ECC）
- **报表查询** — 按区域、物料等维度的统计汇总与 TOP 排名

### 🔧 系统管理
- **用户管理** — 账号创建、角色分配、状态管理
- **角色管理** — 细粒度功能权限配置，支持权限树勾选
- **菜单管理** — 菜单结构、显示顺序和访问权限管理
- **基础信息管理** — SAP 同步的销售组织、工厂、装运点等基础数据查询
- **关联规则管理** — 销售组织→分销渠道→工厂→装运点→产品组 关联维度管理
- **主数据管理** — 客户、物料、价格等核心主数据管理

### 📱 移动端
- 支持 H5 移动端预览（点击顶栏手机图标打开）

## 🎨 特性

- 🌙 支持浅色/深色主题切换
- 📐 响应式布局，适配不同屏幕尺寸
- 🎯 纯前端静态 Demo，无需后端服务
- 💫 流畅的过渡动画与微交互
- 🔍 全局搜索与面包屑导航

## 🛠️ 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- 无框架依赖，一键部署到 GitHub Pages

## 🚀 本地运行

```bash
# 直接打开
open index.html

# 或使用 Node.js 服务器
node server.js
# 访问 http://localhost:3000
```

## 📁 项目结构

```
├── index.html      # 主页面（PC端）
├── h5.html         # 移动端页面
├── styles.css      # 全局样式（含主题变量）
├── app.js          # 应用主控制器（路由/登录/主题）
├── data.js         # Mock 数据
├── pages.js        # 核心业务页面（工作台/订单审核/新建/查询）
├── pages2.js       # 规则页面 & 数据中心页面
├── pages3.js       # 系统管理子页面（用户/角色/菜单/基础信息/关联规则/主数据）
├── server.js       # 简易 Node.js 服务器
└── README.md       # 项目文档
```
